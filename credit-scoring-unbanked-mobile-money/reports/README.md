# Reports

This folder contains the report-ready outputs generated by the notebook workflow.
It is the project's evidence trail: figures, metric tables, calibration artifacts,
threshold analyses, and scored samples that support the narrative in the README and
Streamlit app.

## Figures

Saved plots live in `figures/`. The most important ones cover:

- missingness and target imbalance
- repayment and payment behavior
- baseline model comparison
- calibration and threshold selection
- final model comparison
- deep-learning training diagnostics

## Tables

Saved tables live in `tables/`. They include:

- data access and project planning notes
- validation and data-quality summaries
- exploratory statistics
- proxy-feature summaries
- model comparison tables
- calibration and cost-curve outputs
- final artifact and model-card summaries
- scored inference previews

## Reproducibility note

The repository does not ship with polished final report PDFs. Instead, the notebooks
generate the outputs directly so you can inspect, rerun, and extend the analysis from
source.
